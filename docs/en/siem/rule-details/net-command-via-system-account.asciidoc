=== Net command via SYSTEM account

Identifies attempts to create new users via the SYSTEM account.

*Severity*: low

*Tags*:

* Windows

*Version*: 1

==== Rule query


[source,js]
----------------------------------
process.name: (net.exe or net1.exe) and user.name:SYSTEM
----------------------------------
