[[powershell-spawning-cmd.exe]]
=== PowerShell spawning cmd.exe

Identifies a suspicious parent child process relationship with `cmd.exe`
descending from `PowerShell.exe`.

*Severity*: low

*Tags*:

* Windows

*Version*: 1

==== Rule query


[source,js]
----------------------------------
process.parent.name:powershell.exe and process.name:cmd.exe
----------------------------------
