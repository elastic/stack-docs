=== Potential Evasion via Filter Manager

The Filter Manager Control Program (`fltMC.exe`) binary may be abused by
adversaries to unload a filter driver and evade defenses.

*Severity*: low

*Tags*:

* D-SA
* Windows

*Version*: 1
