= API quick reference

All {dfanalytics} endpoints have the following base:

[source,js]
----
/_ml/data_frame/analytics
----
// NOTCONSOLE

The evaluation API endpoint has the following base:

[source,js]
----
/_ml/data_frame/_evaluate
----
// NOTCONSOLE

[source,js]
----
/_ml/trained_models/
